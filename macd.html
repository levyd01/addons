Multiple Stocks MACD

Introduction

This Google Sheets™ Add-on is useful for investors who want an overview of the MACD on multiple stocks. The MACD Add-on provides Bullish and Bearish MACD crossing events for multiple stocks traded in a single table.

Features

Any stock traded on US Stock Markets can be selected. The user can select the MACD unit interval as well as possibility to modify all internal MACD parameters. The Add-on provides Bearish and Bullish MACD crossing. The user can select to display only simple information or detailed MACD values for each stock.

Usage

You can use this data as Buy/Sell signals or combine it with other indicators for decisions based on combined indicators.

Installation

You can install it via this link. Extensive help is available under 'Add-ons' → 'Multiple Stocks MACD' → 'Help'.
