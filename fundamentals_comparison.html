Stocks Fundamentals Comparison

Introduction

The Stocks Fundamentals Comparison Add-on provides fundamental ratios for selected stocks traded on NYSE or on Nasdaq.

Features

Multiple stocks traded on NYSE or on Nasdaq can be selected. The user can select which fundamental ratios he would like to be displayed. Each of the below fundamental ratios can be selected. They are grouped into common domains of interest:

  • Margins: EBITDA margin, Net margin, Gross margin, Operating margin
  • Valuations: Price to earnings, Price to sales, Price to book, EV/EBIT, EV/Sales, EV/EBITDA
  • Liquidity: Debt to equity, Net debt/EBITDA, Current ratio, Quick ratio
  • Efficiency: Return on assets, Return on equity, Return on invested capital, Return on capital employed, Return on sales
  • Dividend: Dividend yield, Payout ratio

Availability

All features of the Add-on are available for free, immediately after installation. The Add-on can be called 50 times for free. Additional Calls can be purchased on line directly from the Add-on side bar. The Add-on informs the user about the remaining number of Calls.

Accountability

In case the user calls the Add-on but an error occurs, the Call will not be accounted and the remaining number of Calls will not be modified.

Installation

You can install it via this link. Extensive help is available under 'Add-ons' → 'Dividend Comparison' → 'Help'.

'Stocks Fundamentals Comparison' Menu

The 'Stocks Fundamentals Comparison' menu can be found in the Google Sheets™ menu under 'Extensions' → 'Add-ons'. It consists of four elements:

  • 'Stocks Fundamentals Comparison' opens a side bar where the user can select which ratios to retrieve for which stocks.
  • 'Purchase Calls' opens a side bar where the user can check how many Calls are available and select how many Calls to purchase.
  • 'Help' opens this front window.
  • 'About' opens a front window with details about the Add-on version and the author.

'Stocks Fundamentals Comparison' side bar

This sidebar is the main interface of the Add-on. Here the user can select which fundamental ratio to retrieve and for which stocks. The elements of this side bar are:

  • 'Symbol list' to be filled with a comma separated ticker list of a securities traded on the Nasdaq Stock Exchange or the NYSE.
  • 'Margins' with the ratios EBITDA margin, Net margin, Gross margin and Operating margin
  • 'Valuations' with the ratios Price to earnings, Price to sales, Price to book, EV/EBIT, EV/Sales and EV/EBITDA
  • 'Liquidity' with the ratios Debt to equity, Net debt/EBITDA, Current ratio and Quick ratio
  • 'Efficiency' with the ratios Return on assets, Return on equity, Return on invested capital, Return on capital employed and Return on sales
  • 'Dividend' with the ratios Dividend yield and Payout ratio

Press the 'Submit' button to call the Add-on, retrieve the fundamental ratios and display it starting from the current active cell in the current sheet. The 'Reset' button deletes the content of the side bar. The 'Cancel' button closes the side bar without executing the Add-on.

'Purchase Calls' side bar

This sidebar has two purposes: report the number of Calls available and purchase additional Add-on Calls. In order to purchase additional Calls, select how many Calls to purchase. Note the price discount increases with the bigger purchase. Press the 'Proceed' button that opens a new sidebar to select the method of payment. After selecting the method of Payment, a new PayPal front window will open to perform the payment. After a successful transaction, a pop up window will inform about the number of Calls available. The 'Stocks Fundamentals Comparison' side bar will be visible where the user can immediately retrieve new dividend Data using the newly purchased Calls.

Error Messages

Error messages are pop up windows that appear when no dividend Data was retrieved after pressing the 'Submit' button in the 'Stocks Fundamentals Comparison' side bar. The captured types of errors are:

  • Unrecognized symbols. None of the entered symbol is trading neither on the Nasdaq Stock Exchange nor on the NYSE.
  • No more function calls are available. Please purchase additional calls directly in the Google Sheets™ menu under Extensions → Add-ons → Stocks Fundamentals Comparison → Purchase Calls.
